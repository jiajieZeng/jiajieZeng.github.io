CRT

$x \equiv b_1 (mod a_1)$

过程

  • 计算所有模数的积$p$
  • 对于第$i$个方程，计算$r=p/a[i]$
  • 计算$r$在模$a_i$意义下的逆元$r_i^{-1}$
  • 计算$r * r_i^{-1}$不取模
  • 最终答案是$b[i] * r * r_i^{-1}的和$

P1495 【模板】中国剩余定理（CRT）/ 曹冲养猪

模数不互质的情况

假设$$ x \equiv a_1 \pmod {m_1} \ x \equiv a_2 \pmod {m_2}$$

转化为不定方程: $ x = m_1p + a_1 = m_2q + a_2$，其中$p,q$是整数，则有$m_1p-m_2q=a_2-a_1$

由裴蜀定理可知，$a_2-a_1$不能被$gcd(m_1,m_2)$整除时，无解

其他情况下，可以通过$exgcd$求出来一组可行解$p,q$

则原来两方程组的模方程组的解为$x \equiv b \pmod M$，其中$b=m_1p+a_1, M=lcm(m_1, m_2)$
